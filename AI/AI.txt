Three methods of ML : 
-Supervised Learning : labeled data, ex : train AI to classify images to animals (dog, cat, monkey …) 
-Unsupervised Learning : without label, train AI to find herself a natural grouping then classify images 
-Reinforcement Learning : A type of machine learning where an AI learns to accomplish a task by receiving feedback in the form of rewards or penalties for its actions, allowing it to discover the best strategy over time (trial and errors)

AI : simulation de l'intelligence humaine qui inclue : ML, Deep Learning, NLP (lire, comprendre et générer du langage humain), neural network, cognitive computing, computer vision (reconnaissance faciale, lecture dans l'image, video)

ML : branche IA qui permet à la machine d'apprendre à partir des données 
       -apprentissage supervisé : la ML apprend et me retour un résultat attendu (ex : emails --> spam ou pas spam, transactions --> fraude ou pas fraude) :  Classification d'e-mails, Diagnostic médical, Prédiction du prix d'un bien ...
       -apprentissage non supervisé : la ML cherche des structures cachées ou des regroupements ... Segmentation, Trouver des tendances ...
       -apprentissage par renforcement : apprend par essais/erreurs :  Voiture autonome, Trading automatisé

Deep Learning : subset of ML using neural networks : reconnaissance d'image, détection du texte

Generative AI : Special kind of deep Learning that use transformers (encoding, decoding) to create text, image , audio, code …
                 
LLM : un modèle d’intelligence artificielle (IA entrainée sur des données massives) capable de comprendre et de générer du langage humain (texte) à très grande échelle, exemple : GPT-4, claude, Gemini, Mistral, LLaMA

Architecture typique d’un chatbot web

-Frontend (interface utilisateur) :

HTML/CSS/JS ou un framework comme React ou Vue

Affiche les messages utilisateur + bot (style Messenger)

-Backend (serveur) :

En Python avec Flask ou FastAPI

Reçoit les messages, traite la logique, envoie la réponse

-IA / Traitement NLP :

Moteur de règles (simple)

OU modèle IA (comme GPT, BERT, ou un modèle personnalisé)

OU API externe (ex : OpenAI, Google Dialogflow)

-Base de données (optionnelle) :

Sauvegarde les conversations, utilisateurs, préférences, etc.
